# ⚔️ 工場勤務でも世界を救いたい

工場経営 × RPG のブラウザゲームです。インストール不要!

## ▶️ 遊ぶ

**https://sooorai05.github.io/factory-tycoon/**

スマホでもPCでも遊べます。

## 遊び方

### 🏭 工場パート
- クリック / タップ / WASD・矢印キーで移動
- マシンで製品を作り、右の「出荷口」で売却してお金を稼ぐ
- マシン購入・強化、作業員(👷・🦾)の雇用、工場の拡張(全5段階)

### ⚔️ RPGパート
- 稼いだお金で「⚔️ 装備」から武器・鎧・ポーション・攻撃範囲を購入
- 工場の**左のゲート**から「世界を救う旅」へ出発
- すごろく式マップ全10ステージ:草原(1〜3)→ 雪原(4〜7)→ 魔界(8〜9)→ 魔王城(10)
- 敵に近づくと**周囲を範囲攻撃**。弓や魔法で狙ってくる敵は弾を避けよう
- 魔王城では四天王4人を倒すと魔王が出現。**👟移動スピード強化は戦闘の回避にも効く!**
- 戦闘中は Q キー(またはボタン)で回復ポーション

## その他
- 🏆 実績22種(達成で賞金)
- 🏅 ランキング:スコアコードを友達と交換して累計売上で競える
- 💾 10秒ごとに自動セーブ。セーブコードで別の端末への引き継ぎも可能
- ゲームを閉じている間もマシンは生産(効率1/3・最大8時間)
