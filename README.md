# 🏭 ファクトリー・タイクーン

ブラウザだけで遊べる工場経営ゲームです。インストール不要!

## ▶️ 遊ぶ

**https://sooorai05.github.io/factory-tycoon/**

スマホでもPCでも遊べます。

## 遊び方

- クリック / タップ / WASD・矢印キーで社長を移動
- マシンに近づくと製品を自動で回収、右の「出荷口」に行くと自動で売却
- お金でマシンの購入・強化、作業員の雇用、工場の拡張ができます
- 工場を拡張するとスロットが増え、内装がどんどん豪華になります(全5段階)
- 実績は18種類。達成すると賞金がもらえます
- 🏅ランキング:スコアコードを友達と交換すると、同じ画面で累計売上ランキングを競えます

## セーブ

- 10秒ごとに自動セーブされ、次に開くと続きから遊べます
- 「💾 セーブ」→「コードを発行」で出るセーブコードを使えば、別の端末にデータを引き継げます
- ゲームを閉じている間もマシンは生産を続けます(最大8時間分)
